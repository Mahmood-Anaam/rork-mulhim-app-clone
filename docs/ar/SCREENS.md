# مرجع شاشات التطبيق

## وصف الشاشات والتفاعلات

### 1. شاشة الترحيب
**الملف**: `app/welcome.tsx`

```
┌─────────────────────────┐
│                         │
│      🌟 مُلهم           │
│      Mulhim             │
│                         │
│  اختر لغتك              │
│                         │
│  ┌───────────────────┐  │
│  │   العربية         │  │
│  └───────────────────┘  │
│                         │
│  ┌───────────────────┐  │
│  │   English         │  │
│  └───────────────────┘  │
│                         │
└─────────────────────────┘
```

**التفاعلات**:
- النقر على "العربية" → تعيين اللغة العربية، تفعيل RTL، الانتقال للأمام
- النقر على "English" → تعيين اللغة الإنجليزية، الانتقال للأمام
- اختيار اللغة يُحفظ في AsyncStorage

### 3. شاشة تسجيل الدخول
**الملف**: `app/auth/login.tsx`

```
┌─────────────────────────┐
│  → رجوع                 │
│                         │
│   تسجيل الدخول          │
│                         │
│  البريد الإلكتروني      │
│  ┌───────────────────┐  │
│  │                   │  │
│  └───────────────────┘  │
│                         │
│  كلمة المرور            │
│  ┌───────────────────┐  │
│  │ ••••••••          │  │
│  └───────────────────┘  │
│                         │
│  نسيت كلمة المرور؟      │
│                         │
│  ┌───────────────────┐  │
│  │   تسجيل الدخول   │  │
│  └───────────────────┘  │
│                         │
│  ليس لديك حساب؟         │
│  إنشاء حساب             │
└─────────────────────────┘
```

**التفاعلات**:
- إدخال البريد/كلمة المرور → التحقق من المدخلات
- زر "تسجيل الدخول" → يستدعي `signIn()` عبر AuthProvider
- "نسيت كلمة المرور؟" → يحفز `resetPassword()`
- رابط "إنشاء حساب" → ينتقل إلى `/auth/signup`
- يعرض تنبيهات خطأ للبيانات غير الصحيحة

### 4. شاشة التسجيل
**الملف**: `app/auth/signup.tsx`

**التفاعلات**:
- إدخال البريد/كلمة المرور/تأكيد → التحقق من النموذج
- "تسجيل" → يستدعي `signUp()` عبر AuthProvider
- يكشف الحسابات المكررة
- رابط "تسجيل الدخول" → ينتقل إلى `/auth/login`

### ملخص تدفق التنقل

```
الترحيب → اختيار الحساب → المصادقة (دخول/تسجيل) → الإعداد → التبويبات الرئيسية
                                                                      │
                                                    ┌─────────────────┼─────────────────┐
                                                    │                 │                 │
                                              تبويب الخطة      تبويب التغذية     تبويب المدرب     تبويب الملف
                                                    │                 │                             │
                                              تفاصيل التمرين    تفاصيل الوجبة              المعلومات الحيوية
```
