# دليل التشغيل - تطبيق مُلهم

يوفر هذا الدليل تعليمات حول كيفية إعداد وتشغيل تطبيق ملهم محلياً.

## المتطلبات الأساسية
- **Node.js**: الإصدار 18 أو أحدث.
- **Bun**: الإصدار 1.0 أو أحدث (موصى به لإدارة الحزم).
- **Expo CLI**: مثبت عالمياً أو عبر `npx`.
- **حساب Supabase**: لحفظ البيانات.
- **مفتاح OpenAI API**: لميزة المدرب الذكي.

## إعداد البيئة
قم بإنشاء ملف `.env` في المجلد الرئيسي وأضف المفاتيح التالية:

```env
EXPO_PUBLIC_OPENAI_API_KEY='your_openai_key'
EXPO_PUBLIC_SUPABASE_URL='your_supabase_url'
EXPO_PUBLIC_SUPABASE_ANON_KEY='your_supabase_anon_key'
```

## التثبيت
قم بتثبيت حزم المشروع باستخدام Bun:

```bash
bun install
```

## تشغيل التطبيق
يمكنك تشغيل التطبيق باستخدام أوامر Expo القياسية:

### بدء خادم التطوير
```bash
bun run start
```

### التشغيل على محاكي iOS
```bash
bun run ios
```

### التشغيل على محاكي Android
```bash
bun run android
```

### التشغيل على المتصفح (Web)
```bash
bun run web
```

## حل المشكلات
- إذا واجهت أخطاء في التجميع، جرب مسح التخزين المؤقت: `bun x expo start --clear`.
- تأكد من ضبط جميع متغيرات البيئة بشكل صحيح في ملف `.env`.
- لمشاكل الاتصال بـ Supabase، تحقق من رابط المشروع والمفتاح المجهول (Anon Key).
